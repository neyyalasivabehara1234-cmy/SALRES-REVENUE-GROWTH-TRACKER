# Sales & Revenue Growth Tracker in Excel

## Objective
Analyze sales, revenue, and profit trends using Excel dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Dataset
100 rows of sales data across regions, categories, and channels.

## Features
- KPI tracking (Revenue, Profit, Units)
- Growth analysis
- Target vs Achievement
- Interactive dashboard

## How to Use
1. Open Excel file
2. Explore dataset
3. Create Pivot Tables
4. Build dashboard

## Conclusion
This project helps understand real-world business analytics using Excel.
